== Description ==
Slider Bin Plugin allows users to create and manage various types of sliders:
- Hero with Same Heading Slider
- Hero with Separate Heading Slider
- Post Slider
- Image Slider
- Video Slider

Shortcodes are generated for each slider, allowing you to embed sliders anywhere on your WordPress website.

== Features ==
1. **Hero with Same Heading Slider** - Use a single heading, subheading, and button across different images.
2. **Hero with Separate Heading Slider** - Add unique headings, subheadings, and buttons for each image.
3. **Post Slider** - Showcase multiple blog posts with title, excerpt, feature image, and a "Read More" button.
4. **Image Slider** - A standard slider for images.
5. **Video Slider** - Display videos from your media library or embed via YouTube/Vimeo URLs.

== Installation ==
1. Download the plugin zip file.
2. Upload it to your WordPress site via Plugins > Add New > Upload Plugin.
3. Activate the plugin from the Plugins menu.
4. Go to **Sliders** in the admin dashboard to create a new slider.
5. Copy the shortcode generated and paste it into any post, page, or widget to display the slider.

== Frequently Asked Questions ==
= How do I customize the slider style? =
Navigate to **Sliders > Settings** to configure typography, button styles, and other visual aspects.

= Can I use videos and images in the same slider? =
Currently, sliders are designed for one content type per slider. Create separate sliders for images and videos.
